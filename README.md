# YAGL-Translations
[中文](docs/README_zh.md)

This is a collection of OpenTTD NewGRFs with (Simplified Chinese) translations added after distribution.
All NewGRFs collected here are encoded in either m4nfo (stations) or nfo.

## Introduction

OpenTTD is a universal game for everyone, and so are the NewGRFs designed for this game. So the game is designed to be played by anyone, no matter what language they use. However, due to technical issues and lack of translators in the early days, almost all NewGRFs that use m4nfo or nfo don't have proper translations for CJK languages due to lack of support for UTF-8 characters. However, years have passed and now we have [yagl](https://github.com/UnicycleBloke/yagl). Using this tool, we could easily decode NewGRFs (as long as they are not broken) and add translations encoded in UTF-8 - and use them in the game.
