# YAGL-Translations
[English](../README.md)

这是一个收集了在发布后添加了（简体中文）翻译的OpenTTD NewGRFs的合集。
这里收集的所有NewGRFs都以m4nfo（车站）或nfo进行编码。

## 介绍
OpenTTD是为所有人设计的游戏，为此设计的NewGRF也是如此。因此无论是你家的张三还是隔壁的老王，不管说什么语言操什么口音，都应该可以自由游玩OpenTTD。然而，由于早期的技术问题和翻译人手不足，几乎所有使用m4nfo或nfo的NewGRF都没有合适的CJK，即中日韩翻译，这是由于对UTF-8字符的支持不足。不过现在我们有响当当的——[yagl](https://github.com/UnicycleBloke/yagl)。使用这个工具，我们可以轻松解码NewGRF（只要它们没有损坏），并添加以UTF-8编码的翻译，然后在游戏中使用它们。

## 许可证
所有NewGRFs都有自己的许可证，在每个文件夹中都会有原始许可证的副本。原始的变更日志和readme文件（如果有的话）也将包含在其中。
